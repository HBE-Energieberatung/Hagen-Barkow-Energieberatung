# HBE Energieberatung Website

Statische GitHub-Pages-Version der Website von Hagen Barkow Energieberatung.

## Dateien
- `index.html`
- `style.css`
- `assets/logo.png`

## So veröffentlichst du die Seite mit GitHub Pages

1. Dieses Paket entpacken.
2. Alle Dateien in dein Repository hochladen.
3. In GitHub auf **Settings** > **Pages** gehen.
4. Unter **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Speichern.

Danach ist die Seite unter deiner GitHub-Pages-URL erreichbar.
