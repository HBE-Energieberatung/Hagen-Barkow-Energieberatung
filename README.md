# HBE Website – finales Paket v2

## Enthaltene Dateien
- `index.html` – Hauptseite
- `style.css` – CI-Stylesheet
- `danke.html` – Bestätigungsseite nach Formular-Absenden
- `impressum.html` – Impressum (Adresse bereits eingetragen)
- `datenschutz.html` – Datenschutzerklärung (DSGVO-konform)
- `agb.html` – AGB vollständig (13 Paragraphen)

## Vor Veröffentlichung ausfüllen

### Zwingend:
1. **Formspree-ID** in `index.html` ersetzen:  
   `action="https://formspree.io/f/DEINE_FORM_ID"` → echte ID von formspree.io
2. **Platzhalter Impressum** ausfüllen:  
   - Steuernummer
   - USt-IdNr. (falls vorhanden)
   - Name Berufshaftpflichtversicherung

### Optional / sobald vorhanden:
3. **dena-Expertenliste** – Hinweis in Über-mich aktualisieren sobald gelistet
4. **Google Fonts lokal** – für DSGVO-sicherere Alternative Fonts herunterladen

## Deployment auf GitHub Pages
1. Alle Dateien ins GitHub-Repo hochladen (vorhandene Dateien ersetzen)
2. Im Repo: Settings → Pages → Branch: main / root → Save
3. Website erreichbar unter: hbe-energieberatung.github.io/hbe-energieberatung/
